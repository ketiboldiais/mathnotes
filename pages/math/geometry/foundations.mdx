# Primitives

_This note covers basic constructs in geometry._

## Metric Space
> __~definition~.__ Let ${M}$ be a set and let ${d}$ be a function of the form ${d:(M \times M) \mapsto \reals.}$ A _metric space_ is an ordered pair ${(M,d)}$ that satisfies the following axioms, for all ${x,y,z \in M:}$
>
> 1. ${d(x,x)=0.}$
> 2. If ${x \neq y,}$ then ${d(x,y) \gt 0.}$
> 3. ${d(x,y)=d(y,x).}$
> 4. ${d(x,z) \le d(x,y)+d(y,z).}$
> 
## Points
> __~definition~.__ Given a metric space ${(M,d)}$ we call each element of ${M \times M}$ a _point_.
