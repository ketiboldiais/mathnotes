# Reference
_This is a reference of common mathematical propositions._

## Exponents
### Power of Zero
- Iff ${n \gt 0, }$ then ${0^n = 0.}$
- Iff ${n \lt 0,}$ then ${0^n = \text{undefined}.}$
	- Reason: If ${n \lt 0}$ and ${0^{n} = 0,}$ then ${0^{n} = 1/0.}$
- If ${n = 0,}$ then ${0^0}$ is either ${1}$ or undefined.


### Power of One
- ${1^n = 1.}$
- If ${n}$ is even, then ${(-1)^n = 1.}$
- If ${n}$ is odd, then ${(-1)^n = -1.}$


### Recursive Definition of Exponents
$$
b^n = 
\begin{cases}
	1 \space &\text{if} \space n = 0 \\
	b \space &\text{if} \space n = 1 \\
	b \times b^{n-1} \space &\text{else}
\end{cases}
$$

### Addition of Exponents
- ${b^{m+n} = b^m \times b^n.}$

### Negative Exponents
- ${b^{-n} = \dfrac{1}{b^n}.}$
- ${0^{-n} = \text{undefined}.}$

### Associativity of Exponents
- ${(b^m)^n = b^m \times b^n.}$

### Distributivity of Exponents
- ${(ab)^n = a^n \times b^n.}$

### Rational Exponents
- Iff ${n \ge 0,}$ then ${x^{1/n} = \sqrt[n]{x}}$
- ${\sqrt[n]{x} \iff (y^n = x \land y \in \reals^+).}$
- If ${x \in \reals^+}$ and ${\frac{p}{q} \in \mathbb{Q},}$ then
$$
	x^{p/q} = (x^p)^{\frac{1}{q}} = \left(x^{\frac{1}{q}}\right)^p
$$

## Logarithms
### Definition
- ${\log_b(y) = x \iff b^x = y}$ where ${b \neq 0.}$
### Log of One
- Iff ${b \neq 0,}$ then ${\log_b(1) = 0}$ because ${b^0 = 1.}$
- ${\log_b(b) = 1,}$ because ${b^1 = b.}$
### Log of Products
- ${\log_b(xy) = \log_b(x) + \log_b(y).}$
### Log of Quotients
- ${\log_b\left( \frac{x}{y} \right) = \log_b(x) + \log_b(y)}$
	- Reason: ${b^c b^d = b^{c+d}.}$
- ${\log_b(x^d) = d \log_b(x)}$
	- Reason: ${(b^c)^d = b^{cd}.}$
- ${\log_b(\sqrt[n]{x}) = \dfrac{\log_b(x)}{y}}$
	- Reason: ${\sqrt[y]x = x^{1/y}.}$
- ${x^{\log_b(y)}=y^{\log_b(x)}.}$
	- Reason:
		$$
		\begin{aligned}
			x^{\log_b(y)} &= b^{\log_b(x)\log_b(y)} \\
			&= \left(b^{\log_b(y)}\right)^{\log_b(x)} \\
			&= y^{\log_b(x)}
		\end{aligned}
		$$
