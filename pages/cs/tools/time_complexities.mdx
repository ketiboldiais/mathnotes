# Time Complexities

The table below provides an overview of time complexities for various data
structures, algorithms, and library functions in popular languages.

<table>
	<thead>
		<tr>
			<th>Code</th>
			<th>Time Complexity</th>
			<th>Space Complexity</th>
			<th>Language</th>
			<th>Comment</th>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td>`strlen()`</td>
			<td>${O(n)}$</td>
			<td>`~`</td>
			<td>C/C++</td>
			<td>Found in `string.h`, returns the length of a string.</td>
		</tr>
		<tr>
			<td>`push()`</td>
			<td>${O(1)}$</td>
			<td>`~`</td>
			<td>JavaScript</td>
			<td>Array method, adds a new element to the end of an array.</td>
		</tr>
		<tr>
			<td>`pop()`</td>
			<td>${O(1)}$</td>
			<td>`~`</td>
			<td>JavaScript</td>
			<td>Array method, deletes the last element in an array.</td>
		</tr>
		<tr>
			<td>`shift()`</td>
			<td>${O(n)}$</td>
			<td>`~`</td>
			<td>JavaScript</td>
			<td>Array method, deletes the first element in an array.</td>
		</tr>
		<tr>
			<td>`unshift()`</td>
			<td>${O(n)}$</td>
			<td>`~`</td>
			<td>JavaScript</td>
			<td>
				Array method, adds one or more elements at the beginning of an
				array.
			</td>
		</tr>
		<tr>
			<td>`splice()`</td>
			<td>${O(n)}$</td>
			<td>`~`</td>
			<td>JavaScript</td>
			<td>
				Removes, adds, or replaces a new element by index (note: not a very
				well designed library function; code bases using splice tend to be
				buggy).
			</td>
		</tr>
		<tr>
			<td>`sort()`</td>
			<td>${O(n \log n)}$</td>
			<td>`~`</td>
			<td>JavaScript</td>
			<td>Modifies the array by calling a compare function. If the compare function is not provided, the default order is by Unicode value.</td>
		</tr>
		<tr>
			<td>`concat()`</td>
			<td>${O(n)}$</td>
			<td>`~`</td>
			<td>JavaScript</td>
			<td>Creates a new array by merging two or more arrays.</td>
		</tr>
		<tr>
			<td>`slice()`</td>
			<td>${O(n)}$</td>
			<td>`~`</td>
			<td>JavaScript</td>
			<td>Returns a copy of a sub-array between two indices, the start and the end.</td>
		</tr>
		<tr>
			<td>`indexOf()`</td>
			<td>${O(n)}$</td>
			<td>`~`</td>
			<td>JavaScript</td>
			<td>Array method, returns the first index of the argument if found, otherwise returns `-1`.</td>
		</tr>
		<tr>
			<td>`forEach()`</td>
			<td>${O(n)}$</td>
			<td>`~`</td>
			<td>JavaScript</td>
			<td>Array method, executes a function for each array element.</td>
		</tr>
		<tr>
			<td>`for...of`</td>
			<td>${O(n)}$</td>
			<td>`~`</td>
			<td>JavaScript</td>
			<td>Object method, iterates over an object's keys.</td>
		</tr>
		<tr>
			<td>`map()`</td>
			<td>${O(n)}$</td>
			<td>`~`</td>
			<td>JavaScript</td>
			<td>Array method, creates a new array composed of the result of calling callback function argument on each array element.</td>
		</tr>
		<tr>
			<td>`filter()`</td>
			<td>${O(n)}$</td>
			<td>`~`</td>
			<td>JavaScript</td>
			<td>Array method, creates a new array composed of elements that return the call back function true.</td>
		</tr>
		<tr>
			<td>`reduce()`</td>
			<td>${O(n)}$</td>
			<td>`~`</td>
			<td>JavaScript</td>
			<td>Object array method, returns a single value from applying the reduction function for each element.</td>
		</tr>
		<tr>
			<td>`...` (JavaScript spread syntax)</td>
			<td>${O(n)}$</td>
			<td>`~`</td>
			<td>JavaScript</td>
			<td>Expands an array or object into its individual elements or key-value pairs. Frequently used within the `reduce()` method, leading to a time complexity of ${O(n^2).}$</td>
		</tr>
	</tbody>
</table>
