## Binary Search

__~binary search~.__ Let ${k \in \uint}$ be a key, and ${A}$ be the array

$$
  \let{A}{\ix{A_0, A_1, A_2, \ldots, A_{L-1}},} 
$$

where ${L-1 \in \nat}$ is the cardinality, or length, of ${A.}$ Then the function
~search~ returns an element ${m \in A}$ if, and only if, ${m = k.}$ Otherwise,
~search~ returns ${\nil.}$

1. __function__ ~search~(${A}$, ${k}$)
2. ${\let{L}{\card{(A)}}}$
3. ${\let{low}{0}}$
4. ${\let{high}{L}}$
5. ${i \gets \floor{  \dfrac{  low + high  }{ 2 }    }}$
6. __if__ ${k = A_i}$ __then__ __return__ ${k}$
7. __else if__ ${ k \lt A_i  }$ __then__ __set__ ${high \gets i - 1}$
8. __else__ __set__ ${low \gets i + 1}$
9. __if__ ${high \ge low}$ __then__ __goto__ step 4. (~loop~).
