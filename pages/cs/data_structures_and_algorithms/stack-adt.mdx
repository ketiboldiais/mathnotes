# Stacks

_This note provides an overview of stacks_.

## Definitions

> __~stack~.__ Let ${S}$ be a dynamic array of objects: ${\ix{x_1, x_2, \ldots, x_{n-1}, x_n},}$ where ${\len{S}=n.}$ A _stack_ is a structure ${(S,P),}$ where ${P}$ is a set of operations on ${S,}$ defined as follows.

<Grid cols={2}>
<Algo>

__~push~.__ Where ${S}$ is a dynamic array and ${x}$ is an object, ${\df{push}}$ inserts ${x}$ as the last element of the dynamic array.

- __operation__ ${\let{S}{\df{push}~x}}$
	1. __init__ ${\let{i}{\len{S}-1}}$
	2. ${\let{S\ix{i}}{x}}$
	3. __return__ ${S}$

</Algo>
<Algo>

__~pop~.__ Where ${S}$ is a dynamic array, ${\df{pop}}$ removes the last element of ${S.}$ If ${S}$ is empty, then ${\df{pop}}$ returns ${S.}$ Otherwise, ${\df{pop}}$ returns the removed element.

- __operation__ ${{\df{pop}~S}}$
	1. __if__ ${S = \nil}$ __return__ ${S}$
	2. __else__
		1. __init__
			1. ${\let{i}{\len{S}-1}}$
			2. ${\let{x}{S\ix{i}}}$
		2. ${\let{S\ix{i}}{\nil}}$
		3. __return__ ${x}$

</Algo>
</Grid>


~example~. Below is a trace of a program pushing and popping onto a stack.
