## Core Number Theory
> __~definition~.__ Given integers ${a}$ and ${b}$ with ${b \neq 0,}$ we say that ${b}$ divides ${a}$ if, and only if, there exists an integer ${c}$ such that ${ = bc.}$ If ${b}$ divides ${a,}$ then we write ${a \dv b.}$ If ${a}$ does not divide ${b,}$ we write ${a \not\dv b.}$

__~theorem~.__ Let ${a,b,c \in \uint.}$
- If ${a \dv b}$ and ${b \dv c,}$ then ${a \dv c.}$
- If ${a \dv b}$ and ${b \dv a,}$ then ${a = \pm b.}$
- If ${a \dv b}$ and ${a \dv c,}$ then ${a \dv (b+c).}$
- If ${a \dv b}$ and ${a \dv c,}$ then ${a \dv (b-c).}$
